BlockChain数据结构

BlockChain Demo这个网站可以帮助理解底层数据结构：https://andersbrownworth.com/blockchain/block

Transaction Fee

交易费用是支付给区块生产者处理交易的奖励金额，通过以太币或者Gwei支付。

Gas Price

同样是以太币或者Gwei定义，是交易中指定的每单位的Gas成本，价格越高，交易被打包进去区块的机会越大。

to set the cost per unit of gas specified for the transaction

Gas

不能和gas price混淆，Gas指的是执行交易所需要的工作量。而Gas价格这是每单位工作量的成本。

简单的理解就是，当很多人要去交易的时候，特定的区块的空间会变得竞争激烈，此时会提高gas价格以在拥堵的时候限制并去优先的处理。
